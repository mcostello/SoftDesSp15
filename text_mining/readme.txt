THIS PROGRAM SHOULD BE EXECUTED FROM THE TERMINAL. WHEN EXECUTED, THIS CODE WILL PROMPT THE USER TO ENTER TWO CITY NAMES. THE CITY'S NAME MUST BE WRITTEN AS IT WOULD APPEAR ON A WIKIPEDIA URL. SPACES SHOULD BE HYPHENS. IF THE PROGRAM RECIEVES AN INPUT WHICH DOES NOT CORRESPOND TO A WIKIPEDIA ARTICLE, THE PROGRAM WILL RETURN AN ERROR.
